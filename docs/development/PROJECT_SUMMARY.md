## 🎓 学习资源

### 理解核心算法

**日提成计算：**
```python
# 阶梯制提成
1-3单：10元/单
4-5单：20元/单
≥6单：30元/单

# 示例：7单 = 30 + 40 + 60 = 130元
```

**状态流转：**
```
trainee → C → B → A
              ↓   ↓
          eliminated  C（降级）
```

**薪资计算：**
- trainee：仅提成
- C级：min(达标日×30, 90) + 提成
- B级：前6天×88 + 提成
- A级：2200 + 全勤奖(400) + 绩效(0/300/600/1000) + 提成
