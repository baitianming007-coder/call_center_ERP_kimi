## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动服务
```bash
python app.py
```

### 3. 访问系统
```
http://127.0.0.1:8080
```

### 4. 默认账号
- 管理员: `admin` / `123456`
- 经理: `manager` / `123456`
- 财务: `finance` / `123456`
